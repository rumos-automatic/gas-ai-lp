💬 AIは使う人の指示力で結果が変わる

ChatGPTの回答の質は、あなたの指示がどれだけ具体的で明確かによって大きく変わります。 AIはあなたの指示が全てです。 あなたの指示がどれだけ具体的かによって回答結果が変わります。

同じ質問でも、指示の出し方によって、返ってくる回答の質が大きく異なります。 たとえば、「メールを書いてください」という曖昧な指示よりも、 「お客様への謝罪メールを、丁寧で誠実なトーンで書いてください」という具体的な指示の方が、 より良い回答が得られます。

❌ 悪い例 vs ✅ 良い例

❌

悪い例：

メールを書いてください。
✅

良い例：

お客様への謝罪メールを、丁寧で誠実なトーンで書いてください。 商品の発送が遅れたことをお詫びし、今後の対応を明確に伝える内容にしてください。
❌

悪い例：

コードを生成してください。
✅

良い例：

Google Apps Scriptで、Gmailから特定の件名のメールを取得して、 スプレッドシートに転記するコードを生成してください。 メールの件名、送信者、本文、受信日時を記録するようにしてください。

💡 ポイント

指示は「5W1H」（誰が、何を、いつ、どこで、なぜ、どのように）を意識して具体的に伝えましょう。 目的、トーン、形式、制約条件などを明確に指定することで、より精度の高い回答が得られます。

🎓 専門的な内容を出したい時は「あなたは○○の専門家です。」

ChatGPTに専門的な内容を出力させたい場合は、役割を与えることが効果的です。

たとえば、「あなたはマーケティングの専門家です」と最初に伝えることで、 ChatGPTはマーケティングに関する専門的な知識を活用して回答してくれます。 役割を与えることで、回答の質と専門性が大幅に向上します。

役割設定の例

マーケティングの専門家

あなたはマーケティングの専門家です。新商品のプロモーション戦略を提案してください。

→ マーケティング用語を使い、専門的な戦略を提案してくれます

プログラミングの専門家

あなたはプログラミングの専門家です。このコードのエラーを修正してください。

→ エラーの原因を詳しく説明し、修正方法を提示してくれます

ビジネスコンサルタント

あなたはビジネスコンサルタントです。私の事業の課題を分析し、解決策を提案してください。

→ ビジネス的な視点から、戦略的な提案をしてくれます

カスタマーサポート担当者

あなたはカスタマーサポート担当者です。お客様からのクレームに対する返信を作成してください。

→ 顧客対応に適した丁寧で共感的な文章を作成してくれます

✨ 活用のコツ

役割設定は、プロンプトの最初に記載するのが効果的です。 また、「10年以上の経験を持つ」「実績豊富な」などの修飾語を加えると、 さらに専門性の高い回答が得られることがあります。

📚 事前情報を読み込んでおくともっと正確な情報を得れる

ChatGPTに事前情報を与えることで、より正確で具体的な回答を得ることができます。

たとえば、「売上を伸ばす方法を教えてください」という一般的な質問よりも、 「以下は私の会社の商品リストです。[商品リスト貼り付け] この中で売上を伸ばすべき商品を3つ選んでください」 と具体的な情報を提供した方が、より実用的な回答が得られます。

事前情報提供の例

📦 商品リストを提供

プロンプト例：

以下は私が扱っている商品リストです。

1. 商品A：価格3,000円、月間販売数50個
2. 商品B：価格5,000円、月間販売数30個
3. 商品C：価格10,000円、月間販売数10個

この中で売上を伸ばすべき商品を3つ選び、その理由を教えてください。

→ 具体的なデータをもとに、実用的な提案をしてくれます

💬 お客様のフィードバックを提供

プロンプト例：

以下はお客様からのフィードバックです。

「商品は良いが、発送が遅い」
「梱包が丁寧で良かった」
「商品説明がわかりにくい」

これらのフィードバックから改善点をまとめてください。

→ 実際のフィードバックをもとに、具体的な改善策を提案してくれます

🏢 会社情報を提供

プロンプト例：

以下は私の会社の情報です。

業種：オンラインショップ
取扱商品：ハンドメイドアクセサリー
ターゲット：20-30代女性
課題：新規顧客の獲得

売上を上げるための施策を3つ提案してください。

→ あなたの会社に特化した、実践的な施策を提案してくれます

💡 ポイント

事前情報は、できるだけ具体的に提供しましょう。数値データ、固有名詞、状況の詳細など、 具体的な情報が多いほど、より実用的な回答が得られます。

ただし、個人情報や機密情報は含めないように注意してください。

📸 文章で説明しづらい時はスクショを使って状況を送る

複雑な状況や視覚的な情報は、スクリーンショットを使って伝えると効果的です。

たとえば、Webサイトのデザインについて質問したい場合、 「このサイトのヘッダーデザインを改善してください」と文章で伝えるよりも、 実際のサイトのスクリーンショットを添付した方が、より的確なアドバイスが得られます。 また、エラー画面やデータの表なども、スクリーンショットで共有することで、 問題を正確に把握してもらえます。

🪟 Windowsでのスクショ方法

方法1: Snipping Tool

  1. 1. Windowsキー + Shift + S を押す
  2. 2. 画面の一部を選択してキャプチャ
  3. 3. クリップボードにコピーされる
  4. 4. ChatGPTに貼り付け

方法2: Print Screen

  1. 1. Print Screen キーを押す
  2. 2. 全画面がクリップボードにコピー
  3. 3. 画像編集ソフトで必要部分を切り取り
  4. 4. ChatGPTにアップロード

🍎 Macでのスクショ方法

方法1: 選択範囲

  1. 1. Command + Shift + 4 を押す
  2. 2. 画面の一部を選択してキャプチャ
  3. 3. デスクトップに保存される
  4. 4. ChatGPTにアップロード

方法2: 全画面

  1. 1. Command + Shift + 3 を押す
  2. 2. 全画面がデスクトップに保存
  3. 3. 必要に応じて画像を編集
  4. 4. ChatGPTにアップロード

🎓 上級者向けテクニック：思考を広げる質問

より深い洞察や多角的な視点を得るための、上級者向けのフォローアップ質問テクニックを紹介します。 これらを使いこなすことで、ChatGPTから更に価値のある回答を引き出せるようになります。

🧠 1. 思考プロセスを誘導する質問

ChatGPTの思考の方向性を変えることで、新しい視点からの回答を得られます。

• 水平思考で考えてください

• 別の視点から考えてください

• 逆の立場で考えるとどうなりますか？

• この前提が間違っているとしたら？

• 第一原理から考えてください

• クリティカルシンキングで分析してください

🔄 2. 多角的な視点を引き出す質問

賛成だけでなく反対意見も聞くことで、バランスの取れた判断ができます。

• 悪魔の代弁者として反論してください

• 反対意見も教えてください

• メリットとデメリットを両方挙げてください

• リスクと機会の両面から分析してください

• 賛成派と反対派の両方の意見を述べてください

• SWOT分析してください

🔍 3. 深い洞察を得る質問

表面的な回答ではなく、根本原因や長期的な影響まで掘り下げます。

• なぜそうなるのか、根本原因を考えてください

• この方法の前提条件は何ですか？

• 長期的な影響はどうなりますか？

• 二次的、三次的な影響は何ですか？

• この問題の構造を分解してください

• 5回「なぜ？」を繰り返してください

🌈 4. 多様性を求める質問

様々な角度からの回答を得ることで、より包括的な理解が得られます。

• 具体例を3つ、異なる業界から挙げてください

• 初心者・中級者・上級者向けの3パターンで

• 過去・現在・未来の視点でそれぞれ説明してください

• 異なるステークホルダーの視点で考えてください

• 文化的・地域的な違いも考慮してください

• 異なる規模（小・中・大）で比較してください

💼 実践例：上級テクニックを使った対話フロー

テーマ：「新規事業のアイデアを考える」

1回目（基本的な質問）:

新規事業のアイデアを3つ提案してください。

2回目（水平思考）:

水平思考で考えて、常識を疑った全く新しいアプローチはありますか？

3回目（悪魔の代弁者）:

悪魔の代弁者として、これらのアイデアの弱点や失敗する理由を挙げてください。

4回目（長期的影響）:

最初のアイデアを実行した場合、5年後・10年後にどんな影響がありますか？

5回目（多様性）:

異なるステークホルダー（顧客、従業員、投資家）の視点で、それぞれの評価を教えてください。

✅ 結果:

多角的で深い洞察が得られ、単純な「良いアイデア」ではなく、 リスクや長期的影響、様々な立場からの評価を含む包括的な分析が手に入りました。

💡 使い方のコツ

  • • 初心者の方は基本的なフォローアップから始めて、徐々に上級テクニックを試しましょう
  • • すべてのテクニックを一度に使う必要はありません。状況に応じて使い分けてください
  • • 「水平思考」や「悪魔の代弁者」などの用語は、ChatGPTがよく理解しているので積極的に使えます
  • • 複数のテクニックを組み合わせることで、より深い洞察が得られます

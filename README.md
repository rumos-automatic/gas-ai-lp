# AI×GAS実践講座 ランディングページ

## 概要

AI×GAS実践講座のランディングページです。Cloudflare Pagesにデプロイして使用します。
決済はStripe Payment Linksを使用。

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)
- レスポンシブ対応（モバイルファースト）

## デザインシステム

- **背景色**: 白（#FFFFFF）
- **セクション区切り背景**: 薄いグレー（#F9FAFB）
- **メインテキスト**: 濃紺（#1E3A5F）
- **アクセントカラー**: ゴールド（#D4AF37）
- **サブテキスト**: グレー（#6B7280）
- **警告カラー**: 薄い赤（#FEF2F2）
- **成功カラー**: 薄い緑（#F0FDF4）
- **フォント**: Noto Sans JP（Google Fonts）

## ページ構成

### 12セクション

1. **ヒーロー**: メインビジュアル、キャッチコピー、CTA
2. **実績バー**: 累計受講者、満足度、未経験者率
3. **悩み提起**: 3つの悩み
4. **稼げる人と稼げない人の違い**: 比較表形式
5. **解決策（ベネフィット）**: ビフォーアフター形式
6. **選ばれる3つの理由**: カード形式
7. **カリキュラム**: 6ステップのタイムライン
8. **受講者の声**: 3名の感想カード
9. **講師紹介**: プロフィールと実績
10. **料金プラン**: 3つのプランカード
11. **よくある質問**: アコーディオン形式
12. **最終CTA**: 大きなCTAボタン

## 画像について

現在の画像はすべてプレースホルダーです。実際の画像に差し替えてください。

- **hero.png**: ヒーローセクション用（推奨サイズ: 800x500px）
- **problem.png**: 悩み提起セクション用（推奨サイズ: 350x350px）
- **instructor.png**: 講師紹介用（推奨サイズ: 300x300px、丸型）

## Stripe Payment Links設定

`index.html`内の以下のプレースホルダーを実際のStripe Payment Linksに置き換えてください：

- `[STRIPE_LINK_1MONTH]`: 1ヶ月プランのリンク
- `[STRIPE_LINK_3MONTH]`: 3ヶ月プランのリンク
- `[STRIPE_LINK_6MONTH]`: 6ヶ月プランのリンク

### 置き換え方法

```html
<!-- 変更前 -->
<a href="[STRIPE_LINK_1MONTH]" class="pricing-button">このプランで申し込む</a>

<!-- 変更後 -->
<a href="https://buy.stripe.com/xxxxx" class="pricing-button">このプランで申し込む</a>
```

## Cloudflare Pagesへのデプロイ

1. **リポジトリ接続**: Cloudflare Pagesで、GitHubリポジトリを接続
2. **ビルド設定**:
   - ビルドコマンド: 不要（静的サイト）
   - ビルド出力ディレクトリ: `landing-page`
3. **デプロイ**: 自動デプロイが開始されます

### カスタムドメイン設定（オプション）

Cloudflare Pagesの設定画面から、カスタムドメインを追加できます。

## ローカル開発

単純なHTMLファイルなので、以下のいずれかの方法で確認できます：

1. **ブラウザで直接開く**: `index.html`をダブルクリック
2. **ローカルサーバー起動**:
   ```bash
   # Python 3の場合
   python -m http.server 8000

   # Node.jsのhttp-serverを使う場合
   npx http-server -p 8000
   ```

## カスタマイズ

### カラーの変更

`style.css`内のカラー変数を編集してください：

```css
/* 例: アクセントカラーの変更 */
background-color: #D4AF37; /* ゴールド → 別の色に変更 */
```

### フォントの変更

`index.html`の`<head>`内のGoogle Fontsリンクを変更してください。

### セクションの追加・削除

`index.html`のセクションを追加・削除し、`style.css`で対応するスタイルを調整してください。
